About

This is a small, sample Stock Inventory & POS demo application built with Spring Boot and Vaadin. It showcases the core building blocks of a real-world inventory system: products, barcode scanning, stock movements (IN/OUT), and sales tracking.

What you can do

  • Create and manage products (name, barcode, unit, active status)
  • Increase / decrease stock with movement notes (IN / OUT)
  • Scan or enter barcodes in a POS screen to build a cart
  • Complete checkout and record sales totals
  • View a dashboard with KPIs, sales trends, and low-stock alerts (reorder level)

Why it exists

The goal is to provide a clean reference project for learning Vaadin UI patterns and Spring Boot data flows. The code is intentionally simple, easy to extend, and suitable real world usage.
